# 🚀 AgencyFlow - Agency & Staff Management System

AgencyFlow is a web-based management system that allows agencies and event organization companies to manage their field staff, event assignments, work hours, and financial earnings from a single centralized platform.

## ✨ Features
* **Dynamic Dashboard:** Upcoming events, pending payments, and earnings charts integrated with Chart.js.
* **Work Hours (WorkLog) Automation:** Automatic salary/earnings calculation based on hours and days for completed events.
* **Dynamic Theme Engine:** Dark Mode, Soft Pink, and Soft Blue theme options designed using LocalStorage.
* **Object-Oriented Architecture:** Robust database integrity with 1:1 and 1:N relationships between models.

## 🛠️ Tech Stack
* **Backend:** C#, .NET 10 (ASP.NET Core MVC)
* **Database:** SQLite, Entity Framework Core
* **Authentication:** ASP.NET Core Identity
* **Frontend:** HTML5, Razor Views, Bootstrap 5, Tailwind CSS
* **Libraries:** FullCalendar.js, Chart.js, SweetAlert2, Flatpickr
* **Extras:** Python (For visual background processing automation)
